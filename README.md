Flutter-login-app-week1
Week 1 Flutter Internship Task - Login UI with Validation and Navigation

📌 Overview

This project is part of my Flutter Development Internship (Week 1).
It demonstrates basic Flutter concepts including UI design, form validation, and navigation.

✨ Features

* 🔐 Modern Login Screen (Play Store style UI)
* 📧 Email validation using Regex
* 🔑 Password validation (minimum 6 characters)
* 👁 Password visibility toggle
* 🔄 Loading indicator on login
* 📲 Navigation to Home Screen
* 🧩 Clean and structured code

🎨 UI Design

* Gradient background
* Card-based layout (modern UI)
* Icons and proper spacing
* Dashboard-style Home Screen

📂 Project Structure

lib/

* screens/

  * login_screen.dart
  * home_screen.dart
* widgets/
* main.dart

🚀 How to Run

flutter pub get
flutter run
